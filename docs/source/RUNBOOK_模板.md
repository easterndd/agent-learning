# RUNBOOK —— 运维手册（模板）

> 用法：复制这份到自己项目仓库根目录，改名 `RUNBOOK.md`，把【】里的内容换成你项目的实际情况。初始化和后续维护交给 AI 编程工具自动完成更省事。
> 目的：把平时部署、维护、排查会用到的命令记下来，自己以后省事，别人对接/支援时也能直接照着做，不用反复问。
> 用不到的小节可以删；命令尽量写到能直接复制粘贴。
